# Projeto ETL Coinbase

## 📝 Descrição
Este projeto implementa um pipeline ETL (Extract, Transform, Load) que extrai dados de criptomoedas através da API pública da Coinbase, transforma os dados em um formato estruturado e os carrega em um banco de dados para análise posterior.

## 🚀 Funcionalidades
- Extração de dados em tempo real da API da Coinbase
- Coleta de informações sobre preços de criptomoedas
- Transformação e limpeza dos dados
- Armazenamento em banco de dados
- Geração de relatórios analíticos

## 🛠️ Tecnologias Utilizadas
- Python 3.9+
- Requests


## 📋 Pré-requisitos
- Python 3.9 ou superior
- Chave de API da Coinbase
